This is the final working code for CHLOE. 

Test4 is complete in all sense, with the hardware connected and configured. 
testwospeech runs the same system without speech recognition.

You will need internet as the STT is using the Google API. 

servo.csv is the servo lookup for CHLOE's Head and arms. 

names.csv stores the newly learnt/Trained people.

mycsv.csv stores the commands learnt by CHLOE.
